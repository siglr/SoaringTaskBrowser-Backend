# SoaringTaskBrowser-Backend
Repository for the backend-side of the Soaring Task Browser client

## Requirements
- Apache Webserver
- PHP (7.0+)
- MySQL
- phpmyadmin (Optional)

## Endpoints
### getTask
Params: id (int)
Returns: Task with given ID

### getTasks
Params: none
Returns: all Tasks

### search Task
Params: query (string)
Returns: all Tasks that match query.
